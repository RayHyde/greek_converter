Greek Alphabet Converter

Easily convert and read Greek text such as place names. Just type what you see on the special keyboard provided and see the latin equivalent appear. Test yourself with a quiz when you think you know them all. Specially made for mobile devices.

See it in action: http://rayhyde.github.io/greek_converter/
